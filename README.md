# Go Foundations

This repository documents my structured journey of learning Go (Golang) — from basic syntax to core programming concepts and modular design.

## 📚 Topics Covered

* Variables & User Input
* Functions (including multiple return values)
* Packages & Project Structure
* Arrays & Slices
* Conditional Statements (if, else, switch)
* Loops (basic, infinite, range)
* Maps (key-value data structures)
* Structs (custom data types)
* Pointers (memory reference & modification)
* Type Conversion (numeric & string conversions)

## 🎯 Learning Approach

* Concept-based folder structure
* Hands-on code examples for each topic
* Focus on clarity, fundamentals, and clean code

## 🚀 Goal

To build strong Go fundamentals and progress towards advanced topics such as:

* Methods
* Interfaces
* Error Handling
* Goroutines & Concurrency
